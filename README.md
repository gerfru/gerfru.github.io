# gerfru.github.io

Personal portfolio and consulting website for Gerald F. Fruhmann — AI, Compliance & Technology Consulting.

## About

Static single-page site built with plain HTML and CSS. No frameworks, no build step.

Features:
- Bilingual (EN/DE) with client-side language toggle
- Responsive layout
- Open Graph and Twitter Card meta tags for social sharing

## Local development

Open `index.html` in a browser. No server required.
